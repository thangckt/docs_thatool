    plot

    thmd.plot ¶

    This module provides functions and class to produce publication-quality figures. The main idea is to create a handy class which setup all necessary parameters for plotting, and then just call its methods to produce the publication-quality figures.

    The core class is the Plotter, which is built on top of matplotlib.pyplot and pandas.DataFrame.plot. The Plotter, therefore, have all useful functions from both matplotlib.pyplot and pandas.DataFrame.

    Assume we have a data frame df: 

    import pandas as pd
import numpy as np

x = np.arange(0, 1000)
df = pd.DataFrame({'x': x, 'y': x**2})

    Then, there are two ways to use the Plotter to plot the data:

    1. Use as a normal matplotlib.pyplot object. Adapt all functions from matplotlib.pyplot. 
      from thmd.plot import Plotter
plt = Plotter().plt
plt.plot(df['x'], df['y'])
plt.show()
    2. Use as a pandas.DataFrame object. Adapt all keyword arguments from pandas.DataFrame.plot. 
      from thmd.plot import Plotter
p = Plotter(df)
p.plot(x='x', y='y')

    Plotter(df: pd.DataFrame = None, style: str = 'thang') ¶

    Class to produce publication-quality figures with matplotlib.

    Parameters:

    • df (DataFrame, default: None ) –

      input data in form of pd.DataFrame. When no input data, create Plotter object will also sets style for matplotlib.

    • style (str, default: 'thang' ) –

      the plotting style.

    Examples:

    from thmd.plot import Plotter
df = Plotter(df)
df.plot(x='x', y='y')

    plot(**kwargs) -> None ¶

    Plot the figure. Adapt all keywords arguments from pandas.DataFrame.plot() method.

    savefig(file_name: str, **kwargs) -> None ¶

    Save figure to file. Adapt all keywords arguments from matplotlib.pyplot.savefig() method.

    Parameters:

    • file_name (str) –

      path to file. Accept all valid file formats, e.g., *.pdf, *.png, *.jpg, etc.

    Other Parameters:

    • dpi (int = 300) –

      resolution of the figure.

    • bbox_inches (str = 'tight') –

      bounding box in inches.

    • pad_inches (float = 0.05) –

      padding in inches.

    • **kwargs –

      all valid kwargs of matplotlib.pyplot.savefig().

    Restriction

    Must call the plot() method before calling this method.

    Examples:

    df.savefig('test.pdf', dpi=300, bbox_inches='tight', pad_inches=0.05)
